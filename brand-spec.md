# 步天工程社网站品牌规范（v0.3）

## 定位

主页采用“学校社团刊物 + 项目档案”的设计方向。它首先说明社团是谁、做过什么，再提供项目、活动记录和招新入口。视觉应简洁、克制、专业，避免产品发布会式口号，也不借助虚构的太空图形制造气氛。

## 品牌与影像资产

- 机构标识：`static/img/school-logo.png`，杭州第二中学校徽。
- 现有社团图形：`static/img/logo.svg`，在正式品牌身份确认前不作为主视觉放大使用。
- 2024 GFSSM 活动照片与提案截图：`static/img/projects/2024-gfssm/`。
- 活动影像来源：[2024 GFSSM 中国站公开报道](https://news.qq.com/rain/a/20240817A030S200)，原报道注明活动照片由求是创新学院提供。

主页不使用生成图、装饰性 SVG 或 CSS 绘制的太空装置。图片只承担记录与证据作用，配以简短、事实性的说明。

## 色彩

- 机构绿：浅色模式 `#00683D`；深色模式使用提高亮度后的 `#67C991`。
- 纸面、正文、次要文字和边线全部复用全站 `--ap-*` 中性色令牌。
- 主页语义令牌统一以 `--home-*` 命名；机构绿统一由 `--bt-school-green*` 提供。

不使用渐变、透明材质、发光效果或装饰阴影。层级依靠纸面色差、发丝线、字号与间距建立。

## 字体与版式

- 全站使用系统无衬线字体栈：`SF Pro` / `PingFang SC` / `Inter` / 系统字体。
- 主页标题最大 64px，二级标题最大约 39px；不用占满屏幕的海报字。
- 内容宽度上限 1180px，以单列档案表、三列数据区和左右分栏组织信息。
- 项目、工作方向与近期记录采用列表，不使用悬浮卡片。

## 内容页

- 「关于步天」与「加入步天」采用文字和真实活动照片并置的刊物页眉；「项目档案」采用纯文字页眉。
- 内容区上限 1180px，以章节、发丝线、编号表和扁平列表建立层级。圆角只用于照片，不用于信息容器。
- 项目条目按年份、状态、摘要和方向排列；悬停只改变标题与链接箭头，不抬升整个条目。
- 博客与知识库保留 Docusaurus 原有导航、目录、分页和可访问性能力，只统一机构绿、纸面背景和章节排版。
- 面向访客的页面不得出现 `TODO`、测试联系方式、内部源码路径或未经核实的宣传性描述。

## 影像画廊

- 画廊包含现场协作、答辩和提案页面，连续从左向右循环。
- 动效结构参考 [React 官网 `CommunityGallery`](https://github.com/reactjs/react.dev/blob/6be2b020a0cabf2fd6dbff5c42c399b8ac323bca/src/components/Layout/HomeContent.js#L713-L817)：两条相同轨道独立接力，保证循环没有空白或跳变；仅在接近视口时播放。
- 图片统一为 4:3 圆角画幅，奇偶项交替倾斜 1.4°，悬停时轻微回正与放大。
- 活动照片允许按统一比例裁切；提案页面必须完整显示，不能裁掉文字或图表。
- 图片依靠替代文字提供准确说明，画面中不叠加标题或编号。
- 在 `prefers-reduced-motion: reduce` 下停止动画，改为可横向滚动的静态列表，并隐藏重复组。

## 交互

- 仅链接箭头作 180ms 的轻微水平反馈；列表本身不整体移动。
- 不使用滚动劫持、首屏入场动画或滚动驱动的大幅变形。
- 键盘焦点使用机构绿外框；触摸设备上的主要文字链接保留至少 44px 点击高度。
- 所有交互必须在浅色、深色、键盘与移动端环境下可用。
