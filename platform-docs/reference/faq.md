# 常见问题

这里用问答形式回答大家最常遇到的问题。找不到答案时，可以顺着每条里的链接深入，或直接联系社团管理员。

## 登录与账号

### Q：为什么我登不上？

A：常见有三种原因：

- **账号待审批（PENDING）**：注册后需要管理员审批，审批前登录会被引导到等待页。
- **账号被禁用（DISABLED）** 或被拒绝：无法登录，需要联系管理员。
- **密码错误**：核对用户名密码；仍不行请找管理员 **重置密码**。

详见 [注册与审批](../getting-started/register.md) 与 [账号状态说明](../reference/account-states.md)。

### Q：我注册了，为什么还进不去？

A：注册不等于开通。账号需要 **管理员审批** 通过后才能使用；如果一直没通过，可能是管理员还没处理，或平台当前 **关闭了注册**。请联系管理员确认，见 [注册与审批](../getting-started/register.md)。

### Q：忘记密码怎么办？

A：平台 **没有自助找回密码** 功能。请直接联系社团管理员帮你重置，不要反复尝试登录。

### Q：能不能有多个账号？

A：可以。同一浏览器内 **最多保存 3 个账号**，通过顶栏的账号切换器快速切换，不用反复退出登录。详见 [访问与登录](../getting-started/access.md)。

## 权限与后台

### Q：为什么我看不到管理后台？

A：管理后台（账号审批、用户与角色、组别、内容管理、站点设置、操作日志等）只对 **管理员角色** 开放。普通成员看不到属于正常现象。角色划分见 [角色与权限](../reference/roles-permissions.md)。

### Q：我应该有某个功能，但打不开？

A：多数功能受权限码约束（例如成员目录的 `members:view`、工作日志的 `worklogs:view`）。缺权限时对应入口会不可见或打不开，可联系管理员核实，参考 [角色与权限](../reference/roles-permissions.md)。

## 预览版与数据安全

### Q：登录页的「预览版」点了能干嘛？

A：那是一个 **免注册的演示沙盒**。不用账号就能进去随便点，体验平台的各个模块。它与正式环境 **完全隔离**，你在里面的任何操作都不影响真实数据。详见 [预览模式](../getting-started/preview.md)。

### Q：预览会不会污染正式数据？我的数据安全吗？

A：不会。预览沙盒是 **完全隔离** 的独立环境，里面的增删改不会流入正式数据，正式数据也不会因为别人玩预览而改变。放心体验，见 [预览模式](../getting-started/preview.md)。

## 使用与设置

### Q：手机能用吗？

A：可以。平台 **桌面端优先**，但同时兼容手机和平板，用手机浏览器即可打开 [https://gfssm.butian.club](https://gfssm.butian.club)。复杂操作（如编辑长文档）在电脑上体验更好。

### Q：怎么改主题和语言？

A：在 **外观设置** 里改。主题可选 浅色 / 深色 / 跟随系统，语言可选 中文 / 英文，还能自定义品牌主色。个人中心和登录页右上角都有外观控件，见 [个人中心与外观](../guide/profile.md)。

### Q：为什么我没收到通知？

A：平台通知是 **纯站内** 的，不发邮件、不发短信。请回到平台看顶栏的铃铛图标是否有红点和未读计数；能收到哪些事件取决于你是否是相关人。详见 [通知中心](../guide/notifications.md)。

### Q：在哪里记录我每天做了什么？

A：在个人中心（/profile）的 **工作日志** 里，用 Markdown 记录，可编辑可删除。见 [工作日志](../guide/worklogs.md)。

### Q：怎么找到某位同伴、看他在不在线？

A：打开 [成员目录](../guide/members.md)（/members），可以查看成员资料、组别、角色和在线状态（在线 / 忙碌 / 离开 / 专注 / 离线）。
