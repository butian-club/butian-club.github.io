# 成员目录

成员目录让你看到 **社团里都有谁**：他们的资料、所在组别、是管理员还是普通成员，以及此刻是否在线。想找人对接、想知道某个头衔背后是谁，从这里开始。

- 👥 成员列表：[https://gfssm.butian.club/members](https://gfssm.butian.club/members)
- 👤 成员详情：`https://gfssm.butian.club/members/[id]`（点列表里的成员进入）

查看成员目录需要 `members:view` 权限。

## 成员列表

打开 **/members** 会看到全体成员的列表。你可以在这里快速浏览，并点击任意成员进入其详情页。

1. 打开 [https://gfssm.butian.club/members](https://gfssm.butian.club/members)。
2. 浏览成员列表。
3. 点击某位成员，进入其 **详情页（/members/[id]）** 查看更多信息。

## 成员详情能看到什么

在成员详情页，你可以了解一位成员的公开信息：

| 信息 | 说明 |
| --- | --- |
| **真实姓名** | 成员填写的真实姓名 |
| **昵称** | 平台内常用的称呼 |
| **头衔** | 例如「结构组组长」「热控工程师」等 |
| **简介** | 成员自我介绍 |
| **性别展示** | 按成员自己的设置展示（不透露 / 男 / 女 / 自定义文本） |
| **所在组别** | 成员归属的组 |
| **角色** | **管理员** 或 **成员** |
| **在线状态（presence）** | 在线 / 忙碌 / 离开 / 离线 / 专注 |

:::note[资料由成员自己维护]
这些资料是成员在 [个人中心](./profile.md) 里填写的。看到的性别展示形式，也由对方自己选择（可以选择不透露）。想修改你自己的资料，去个人中心即可。
:::

## 在线状态怎么理解

在线状态（presence）反映成员当前的可联系程度，常见有下面几种：

| 状态 | 含义 |
| --- | --- |
| **在线** | 正在使用平台，通常可以联系 |
| **忙碌** | 在线但不希望被打扰 |
| **离开** | 暂时不在 |
| **专注** | 正在专注工作，尽量少打扰 |
| **离线** | 未在使用平台 |

:::tip[找人前先看状态]
需要即时对接时，优先找 **在线** 的成员；看到 **忙碌 / 专注** 时，考虑用工单或评论留言，让对方有空再回。
:::

## 查看 vs 管理

成员目录默认是 **只读** 的：你能看资料、组别、角色和状态，但不能修改别人的账号。

| 能力 | 谁可以 | 权限 |
| --- | --- | --- |
| 浏览成员、查看详情 | 拥有查看权限的成员 | `members:view` |
| 改角色、禁用账号等管理操作 | 管理员 | `members:manage` |

:::info[需要改某人的角色或禁用账号？]
这些属于 **管理员** 的能力，在管理后台完成，见 [用户与角色管理](../admin/approvals-users.md)。关于角色划分，见 [角色与权限](../reference/roles-permissions.md)。
:::

## 常见问题

| 问题 | 说明 |
| --- | --- |
| 打不开成员目录？ | 可能缺少 `members:view` 权限 |
| 想改别人的组别 / 角色？ | 需要管理员，见 [用户与角色管理](../admin/approvals-users.md) |
| 为什么看不到某人性别？ | 对方可能选择了「不透露」 |
| 怎么修改我自己的资料？ | 去 [个人中心](./profile.md)（/profile） |
