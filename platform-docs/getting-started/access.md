# 访问与登录

本页介绍如何打开平台、完成登录、在同一浏览器里切换多个账号，以及登录页上的外观设置。第一次接触平台，建议先读 [协作平台简介](../intro.md)。

## 打开平台

平台是一个中文界面、桌面端优先（同时兼容手机与平板）的团队后台，用浏览器即可访问，无需安装任何客户端。

- 🌐 访问地址：[https://gfssm.butian.club](https://gfssm.butian.club)
- 🔑 登录页面：[https://gfssm.butian.club/login](https://gfssm.butian.club/login)

推荐使用较新版本的 Chrome、Edge、Safari 或 Firefox。

:::tip[只想看看它长什么样？]
不想注册也能体验：在登录页点 **「🧪 预览版」**，即可进入一个隔离的演示沙盒，随便点、不影响真实数据。详见 [预览模式](./preview.md)。
:::

## 登录

平台使用 **用户名 + 密码** 登录（基于 Auth.js v5 的凭据登录）。

1. 打开 [https://gfssm.butian.club/login](https://gfssm.butian.club/login)。
2. 输入你的 **用户名** 和 **密码**。
3. 点击登录按钮。
4. 登录成功后进入 **工作台（首页）**，即可开始使用。

:::note[还没有账号？]
平台账号需要先 **注册 → 管理员审批 → 激活** 才能使用。请参阅 [注册与审批](./register.md)。
:::

### 登录后可能出现的情况

账号有不同的状态，登录后的走向也不同：

| 账号状态 | 登录结果 |
| --- | --- |
| **ACTIVE（已激活）** | 正常进入工作台，可使用全部对应权限的功能 |
| **PENDING（待审批）** | 登录后被引导到 `/pending` 等待页，需等待管理员审批 |
| **REJECTED（已拒绝）** | 无法登录 |
| **DISABLED（已禁用）** | 无法登录 |

各状态的详细含义与流转，见 [账号状态说明](../reference/account-states.md)。

## 多账号切换

同一台电脑上如果有多个身份（例如同时是普通成员，又临时借用管理员账号处理事务），不必反复退出再登录。

- 平台支持在 **同一浏览器** 内保存并切换 **最多 3 个账号**。
- 通过顶栏的 **账号切换器** 即可在已保存的账号之间快速切换。

:::info[小提示]
如果你需要保存第 4 个账号，请先移除一个不再使用的账号，或改用浏览器的另一个配置文件 / 隐身窗口登录。
:::

## 登录页外观设置

登录页 **右上角** 有一个「外观」控件，可以在登录前就把界面调成你喜欢的样子。设置项包括：

| 设置项 | 可选值 |
| --- | --- |
| **主题** | 浅色 / 深色 / 跟随系统 |
| **品牌色** | 切换界面主色调 |
| **语言** | 中文 / 英文 |

这些偏好会影响你看到的界面外观，可随时调整。

## 忘记密码怎么办

平台 **没有自助找回密码** 的功能。

:::warning[找管理员重置]
如果忘记了密码或无法登录，请直接联系社团管理员帮你 **重置密码**，而不是反复尝试登录。
:::

## 常见登录问题

| 现象 | 可能原因与处理 |
| --- | --- |
| 提示用户名或密码错误 | 核对输入；注意大小写；仍不行请找管理员重置密码 |
| 登录后停留在等待页 | 账号处于 **PENDING**，等待管理员审批，见 [注册与审批](./register.md) |
| 完全无法登录 | 账号可能是 **REJECTED / DISABLED**，请联系管理员，见 [账号状态说明](../reference/account-states.md) |
| 想换个身份操作 | 使用顶栏 **账号切换器**（最多保存 3 个账号） |
