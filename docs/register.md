# 注册与审批

平台账号采用「**注册 → 管理员审批 → 激活**」的流程。也就是说，注册成功并不代表马上就能用，还需要管理员审批通过。本页说明完整流程和常见问题。

## 注册流程一览

| 步骤 | 你做什么 | 账号状态 |
| --- | --- | --- |
| 1. 提交注册 | 打开注册页，填写信息并提交 | → **PENDING（待审批）** |
| 2. 等待审批 | 在 `/pending` 等待页等待 | **PENDING** |
| 3. 管理员处理 | 管理员在后台审批 | 通过 → **ACTIVE** / 拒绝 → **REJECTED** |
| 4. 正常登录 | 审批通过后即可登录使用 | **ACTIVE（已激活）** |

各状态的完整含义见 [账号状态说明](./account-states.md)。

## 第一步：填写注册信息

1. 打开注册页：[https://gfssm.butian.club/register](https://gfssm.butian.club/register)
2. 填写以下信息：

| 字段 | 说明 |
| --- | --- |
| **用户名** | 你的登录名 |
| **邮箱** | 你的邮箱地址 |
| **密码** | 用于登录的密码 |

3. 表单会做 **前端校验**（基于 Zod），若格式不符合要求会即时提示，请按提示修正后再提交。
4. 点击提交。

:::tip[填写建议]
用真实、能被认出的用户名，方便管理员在审批时确认你的身份，加快通过速度。
:::

## 第二步：等待审批

提交成功后，你的账号即为 **PENDING（待审批）** 状态，页面会进入 `/pending` 等待页。

- 此时账号 **还不能正常使用**。
- 如果你在这个阶段登录，也会被引导回等待页。
- 请耐心等待管理员审批；必要时可主动联系管理员催办。

## 第三步：管理员审批

管理员会在**成员目录**页面（`/members`）处理你的申请，结果有两种：

| 审批结果 | 账号状态 | 之后 |
| --- | --- | --- |
| **通过** | → **ACTIVE（已激活）** | 可以正常登录使用平台 |
| **拒绝** | → **REJECTED（已拒绝）** | 无法登录 |

审批通过后，就可以按 [访问与登录](./access.md) 的说明登录了。

:::note[管理员视角]
关于审批操作的细节，管理员可参阅 [成员目录](./members.md) 的「审批与管人」。
:::

## 关于注册开关

是否开放注册、以及注册是否需要审批，由 **管理员在站点设置中控制**（详见 [站点设置](./settings.md)）。

:::warning[找不到注册入口？]
如果管理员 **关闭了注册**，注册入口将不可用。这种情况下，请直接联系管理员为你创建账号或开放注册。
:::

## 注册限流

为了防止恶意刷注册，注册接口按 **IP 地址限流**：

- 每个 IP **每小时最多 10 次** 注册请求。

正常使用几乎不会触及这个上限。如果你短时间内多次提交并被限流，稍等一会儿再试即可。

## 常见问题

| 问题 | 解答 |
| --- | --- |
| 注册后能马上登录吗？ | 不能，需等待管理员审批通过（账号变为 ACTIVE）后才能正常使用 |
| 一直卡在等待页怎么办？ | 说明仍是 PENDING，请联系管理员催办审批 |
| 找不到注册入口？ | 可能是管理员关闭了注册，请联系管理员 |
| 被拒绝了怎么办？ | 账号变为 REJECTED、无法登录，可联系管理员了解原因 |
| 忘记密码？ | 平台没有自助找回，请找管理员重置，见 [访问与登录](./access.md) |
