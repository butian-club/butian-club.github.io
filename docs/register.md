# 注册与邀请码

平台注册采用 **邀请码制**：你需要一个由管理员发放的 **邀请码** 才能注册。填写信息、
填入有效邀请码并提交后，账号 **立即激活、可直接登录使用**——**没有** 管理员逐个审批的环节。

:::tip[一句话]
拿到邀请码 → 填注册表 → 提交即用。没有「等待审批」这一步。
:::

## 注册流程一览

| 步骤 | 你做什么 | 结果 |
| --- | --- | --- |
| 1. 拿到邀请码 | 向管理员索取一个邀请码 | 得到一串邀请码 |
| 2. 填写注册信息 | 打开注册页，填资料 + 邀请码 | — |
| 3. 提交 | 系统校验邀请码是否有效 | 有效 → 账号 **ACTIVE（已激活）** |
| 4. 直接登录 | 用刚注册的账号密码登录 | 正常进入工作台 |

账号状态的完整含义见 [账号状态说明](./account-states.md)。

## 填写注册信息

打开注册页：[https://gfssm.butian.club/register](https://gfssm.butian.club/register)，填写：

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| **用户名** | 是 | 你的登录名 |
| **真实姓名** | 是 | 便于队友与管理员认出你 |
| **邮箱** | 是 | 你的邮箱地址 |
| **昵称** | 否 | 展示用的别名，不填则用真实姓名 |
| **密码** | 是 | 用于登录 |
| **邀请码** | 是 | 管理员发放，不区分大小写 |

表单会做 **前端校验**（基于 Zod），格式不符会即时提示，按提示修正后再提交。

## 邀请码是什么、从哪来

邀请码由 **管理员在 [成员目录](https://gfssm.butian.club/members) 页面生成** 并发给你。一个邀请码可以带有这些限制：

- **可用次数**：用完就失效（例如一个码给一批人共用，或一人一码）。
- **有效期**：过期后不能再用。
- **预绑定部门**：如果邀请码预先绑定了某个部门，你用它注册后会 **自动加入该部门**，无需再手动分配。
- **可被停用**：管理员随时可以停用（撤销）一个邀请码。

所以，注册时若提示邀请码无效 / 过期 / 已用完 / 已停用，请向管理员索取一个新的可用邀请码。

## 关于注册开关

是否开放公开注册由 **管理员在站点设置中控制**（详见 [站点设置](./settings.md)）。

:::warning[找不到注册入口 / 提示已关闭注册？]
如果管理员 **关闭了公开注册**，注册页会提示「当前已关闭公开注册」。这种情况下请直接联系管理员，
让其开放注册或为你创建账号。
:::

## 注册限流

为防止恶意刷注册，注册接口按 **IP 地址限流**：每个 IP **每小时最多 10 次** 注册请求。
正常使用几乎不会触及。若短时间内多次提交被限流，稍等片刻再试即可。

## 常见问题

| 问题 | 解答 |
| --- | --- |
| 注册后能马上登录吗？ | 能。邀请码有效即激活（ACTIVE），无需等待审批 |
| 没有邀请码怎么办？ | 向管理员索取；没有邀请码无法注册 |
| 邀请码提示无效 / 过期 / 已用完？ | 该码不可用，请找管理员要一个新的 |
| 找不到注册入口？ | 可能是管理员关闭了公开注册，请联系管理员 |
| 用邀请码注册后进哪个部门？ | 若邀请码预绑定了部门则自动进入；否则由管理员后续分配 |
| 忘记密码？ | 平台没有自助找回，请找管理员重置，见 [访问与登录](./access.md) |
