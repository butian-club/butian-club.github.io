# 个人中心与外观

个人中心（/profile）是你自己的「设置页」：在这里维护个人资料、调整界面外观，也在这里写工作日志。你填写的资料会展示在 [成员目录](./members.md) 里，方便同伴认识你。

- ⚙️ 个人中心地址：[https://gfssm.butian.club/profile](https://gfssm.butian.club/profile)

## 编辑个人资料

在个人中心可以维护下面这些资料：

| 字段 | 说明 |
| --- | --- |
| **真实姓名** | 你的真实姓名 |
| **昵称** | 平台内常用的称呼 |
| **头衔** | 例如「结构组组长」「热控工程师」 |
| **简介** | 自我介绍，让同伴快速了解你 |
| **手机** | 联系方式 |
| **性别展示** | 选择 **不透露 / 男 / 女 / 自定义文本** |

修改步骤：

1. 打开 [https://gfssm.butian.club/profile](https://gfssm.butian.club/profile)。
2. 找到 **个人资料** 区域，编辑对应字段。
3. 保存，资料随即更新。

:::tip[性别展示可以「不透露」]
性别展示完全由你决定：可以选择 **不透露**，也可以填 **自定义文本**。别人在 [成员目录](./members.md) 里看到的，就是你在这里的设置。
:::

## 外观设置

外观设置决定界面长什么样，随时可调，只影响你自己看到的界面。

| 设置项 | 可选值 |
| --- | --- |
| **主题模式** | 浅色 / 深色 / 跟随系统 |
| **品牌主色** | 自定义颜色，切换界面主色调 |
| **界面语言** | 中文 / 英文 |

:::note[外观在登录前也能调]
登录页右上角也有「外观」控件，可以在登录前就设置好主题、品牌色和语言，见 [访问与登录](./access.md)。登录后，这些偏好在个人中心同样可以随时修改。
:::

## 在线状态

平台像 QQ 一样有**在线状态**，让同伴一眼看出你此刻方不方便联系。它**不在个人资料里设置**，
而是在**顶栏**（页面右上角你的头像旁）随手切换：

| 状态 | 颜色 | 含义 |
| --- | --- | --- |
| **离线** | 灰 | 你没在使用平台时**自动**显示，不用手动设置 |
| **在线** | 绿 | 正常在线，通常可以联系 |
| **空闲** | 灰白 | 在线但暂时离开一会儿 |
| **工作中** | 蓝 | 正在专注推进事情 |
| **忙碌** | 黄 | 在线但不希望被打扰 |

- **离线是自动判定的**：一旦你离开平台一段时间，状态会自动变灰，无需操心。
- 你在线时，可以从上面四种里**自己选**一个，告诉大家现在的状态。

别人会在哪看到？就在 [成员目录](./members.md) 和 [聊天](./chat.md) 的成员头像上。

:::tip
埋头赶工不想被打扰，就把状态设成 **忙碌** 或 **工作中**；同伴看到会更倾向于用
[聊天](./chat.md) 留言而不是即时催你。
:::

## 写工作日志

个人中心也是你 **记录工作日志** 的地方。工作日志用 Markdown 记录你每天做了什么，可编辑、可删除，属于个人内容。

- 完整用法见 [工作日志](./worklogs.md)。

## 修改密码与账号相关

平台对密码和账号相关操作 **不一定提供自助入口**。

:::warning[没有自助项就找管理员]
如果个人中心里没有提供修改密码等自助选项，请直接联系社团管理员处理（例如重置密码）。关于账号状态与流转，见 [账号状态说明](./account-states.md)。
:::

## 常见问题

| 问题 | 说明 |
| --- | --- |
| 改了资料别人多久能看到？ | 保存后即更新，展示在 [成员目录](./members.md) |
| 不想公开性别？ | 性别展示选 **不透露** 即可 |
| 主题 / 语言在哪改？ | 个人中心的 **外观设置**，或登录页右上角外观控件 |
| 怎么改密码？ | 若无自助项，联系管理员重置 |
| 工作日志在哪写？ | 就在个人中心，见 [工作日志](./worklogs.md) |
