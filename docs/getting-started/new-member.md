# 新成员上手指南

欢迎加入步天工程社。这份指南帮你在最初的几周里找到节奏。

## 你的第一周

1. **读完知识库的入门部分。** 也就是本指南和[工具链](./toolchain.md)，
   再快速浏览一遍[安全规范](../standards/safety.md)。
2. **认识你的代表队 / 项目组。** 你会被分到一个在研方案或新成员练习项目，
   先了解它在做哪个赛题、目前进展到哪、对应哪个部门
   （管理 / 结构 / 人居 / 运营 / 基础设施）。
3. **领一个小任务。** 从一个边界清晰、能在一两周内完成的小任务开始，
   比如读一段 RFP、整理一个参数、画一张示意图——动手比反复观望更有效。

## 我们的协作节奏

- **例会**：每周固定一次，同步进度、暴露问题、安排下一步。

- **记录**：重要的决定、关键参数和踩过的坑，都记到项目文档里。
- **复盘**：一次集训、一场答辩或一个阶段结束后，一起回顾哪里做得好、哪里要改。

## 一些建议

- **不会很正常。** 没有人一开始就什么都懂，关键是愿意查、愿意问、愿意试。
- **先查再问。** 自己先查资料、动手验证，再带着具体问题来讨论，效率会高很多。
- **小步快跑。** 把大目标拆成能验证的小步，每一步都留下可检查的结果。
