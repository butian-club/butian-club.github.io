- **演示文稿**：PPT / Keynote（TODO：确认社团模板与规范）。

## 估算与仿真

很多设计决策要靠量化估算支撑，而不是拍脑袋。

- **编程语言**：Python（数据处理、估算、画图、轻量仿真）。
- **电子表格**：做生命保障、营养与农业、结构、配重、轨道与能量等参数估算与预算表。
- **常用库**：NumPy / Matplotlib 等（按需要使用）。
- **仿真**：按赛题选择（轨道、结构、热控、人口模型等），具体工具与假设在项目文档中说明。

## 环境约定

为减少「在我电脑上能跑」的问题，我们约定：

- 脚本与计算尽量留下依赖说明和输入假设，让别人能复现你的数字。
- 大文件（渲染图、视频、完整 PPT）不要直接塞进代码仓库，按归档规范处理。

:::
