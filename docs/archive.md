# 资料归档

为了让做过的东西能被后来的人找到，我们把可复用的资料集中归档。
这一页是索引，告诉你东西放在哪、怎么放。

## 归档什么

- **方案资料**：赛题（RFP）解读笔记、方案 PPT、效果图、参数与预算表、复盘文档。
- **答辩资料**：英文答辩稿、评委问答记录、排练复盘。
- **学习资料**：各部门的入门笔记、参考链接、历年获奖方案的拆解。
- **活动记录**：比赛、集训、科普公益活动的总结（也会发布到[活动记录](/blog)）。

## 怎么归档

- **大文件**（完整 PPT、渲染图、视频等）不直接放进代码仓库，
  统一存到社团的资料空间。
  > TODO：填写社团资料存放位置（网盘 / 服务器 / 仓库 LFS 等）。
- **每份资料配一句话说明**：是什么、属于哪个赛题 / 项目、什么时候做的。
- **命名清晰**：用 `赛题-内容-日期` 这类一眼能看懂的格式。
