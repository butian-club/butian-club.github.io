# 附件管理

**附件管理**是后台里为数不多仍然保留的「内容」板块，用来**集中查看和清理全站上传的
附件**——任务、工单、文档等处上传的文件都汇总在这里，方便统一盘点、控制存储占用。

- 📎 访问地址：[https://gfssm.butian.club/admin/assets](https://gfssm.butian.club/admin/assets)

:::note[任务 / 工单 / 文档 / 公告的管理已经搬走]
它们不再有独立的后台页面，管理动作都**内联在各自的功能页面**里完成。请到：
[任务](./tasks.md)、[工单](./tickets.md)、[文档](./documents.md)、
[公告](./announcements.md)（在首页发布）。总览见 [管理后台总览](./overview.md)。
:::

## 能做什么

| 动作 | 说明 |
| --- | --- |
| **查看** | 浏览全站已上传的附件与资源列表 |
| **清理** | 删除不再需要的附件，回收存储空间 |

:::caution[删除前先确认没人在用]
删除附件可能影响引用它的任务、文档等。删除前请确认该附件不再被使用——它是**不可轻易
撤销**的动作。
:::

## 附件的访问是受控的

日常上传/下载附件发生在各功能页面（任务、工单、文档详情页等）。这些附件**不是公开的**：

- 必须**登录**、且在**允许的范围**内才能下载，不会对外泄露。
- 账号未激活会受限，见 [账号状态](./account-states.md)。
