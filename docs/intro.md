# 知识库总览

这里是步天工程社的内部知识库，用来沉淀我们在太空城市 / 太空基地设计竞赛中
积累的方法、规范与资料。我们以 ISSDC（国际太空城市设计大赛）、GFSSM
（全球未来太空学者大会）这类系统级设计竞赛为主线，按虚拟航天公司的方式分工，
做方案设计、答辩与航天科普。
它面向两类人：

- **新成员**：快速了解社团怎么运转、用什么工具、要遵守哪些规范。
- **老成员**：在做项目时随手查阅，并把新的经验补充进来。

我们的原则很简单：**做过的事，尽量写下来。** 一份清楚的文档，
往往比一次口头讲解能帮到更多人，也能传给下一届。

## 怎么用这份知识库

- 第一次来，建议按下面的顺序读：

  1. [新成员上手指南](./getting-started/new-member.md) —— 了解社团节奏与你的第一步。
  2. [工具链](./getting-started/toolchain.md) —— 做方案设计与答辩常用的工具。
  3. [项目规范](./standards/project-standards.md) —— 一个方案怎么立项、推进、归档。
  4. [安全规范](./standards/safety.md) —— **外出集训与动手前必读。**

- 需要查资料时，去 [资料归档](./archive.md)。

## 怎么补充内容

知识库的内容存放在仓库的 `docs/` 目录，使用 Markdown / MDX 编写。
如果你在项目中踩过坑、总结出更好的做法，欢迎补充或修订对应文档——
这正是它能持续有用的原因。
