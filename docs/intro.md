# 协作平台简介

**步天工程社协作平台**是社团在比赛季用来管理人、任务、工单、文档与比赛进度的
内部协作系统。它是一个中文界面、桌面端优先（同时兼容手机与平板）的团队后台，
你可以把它理解为社团自建的「迷你飞书 / 钉钉」。

- 🌐 **访问地址**：[https://gfssm.butian.club](https://gfssm.butian.club)
- 🏠 **社团官网**：[butian.club](https://butian.club)（你现在看的这份文档就挂在官网上）
- 🧪 **免注册体验**：登录页点 **「预览版」** 即可进入隔离的演示沙盒，随便点、不影响真实数据。

:::tip[第一次来？]
如果你只是想快速看看它长什么样，直接打开
[gfssm.butian.club/login](https://gfssm.butian.club/login) → 点 **「预览版」**。
详见 [预览模式](./preview.md)。
:::

## 它能做什么

| 模块 | 一句话说明 |
| --- | --- |
| **聊天** | 公司总群 / 开放论坛 / 部门频道 / 私聊，@提及、投票、接龙报名，还能把消息一键转成任务·工单·公告 |
| **任务** | 派活、跟进展、传附件，按用户或部门指派 |
| **工单** | 提资源申请 / 问题反馈，管理员用 Markdown 评论处理 |
| **文档** | 沉淀规范、手册、资料，支持 Markdown 分栏编辑 |
| **公告** | 发布通知，可置顶并展示在首页，自动播报到公司总群 |
| **部门** | 按分工把成员分组，任务可整组指派、聊天自动建群 |
| **成员目录** | 查看社团成员、部门、角色与在线状态 |
| **通知中心** | 任务 / 工单 / 评论等事件自动提醒，带未读计数 |
| **比赛倒计时** | 配置目标日期，首页自动倒数 |
| **工作日志** | 记录每天做了什么，个人可编辑 |

每个成员还有 QQ 式的**在线状态**（在线 / 空闲 / 工作中 / 忙碌，离线自动），
以及一个只在聊天里用 `@ai` 呼叫的 **AI 助手**。

管理能力**分散在各功能页面**——管理员在任务、成员、部门、文档等页面里**直接操作**，
不用专门跑后台。**管理后台**只保留系统级设置：站点设置、比赛倒计时、附件与操作日志。
详见 [管理员指南](./overview.md)。

## 谁能用、怎么进

- 平台账号采用**邀请码注册**：拿到管理员发放的邀请码，填表提交即**立即激活可用**，没有审批等待，详见
  [注册与邀请码](./register.md)。
- 登录后按角色（**管理员 / 成员**）看到不同的功能范围，详见
  [角色与权限](./roles-permissions.md)。
- 支持在同一浏览器里**切换多个账号**（最多 3 个），详见
  [访问与登录](./access.md)。
