# 聊天

**聊天**是平台的即时沟通中心：公告在这里同步、想法在这里冒出来、活儿在这里对齐。
它是实时的——别人发的消息会立刻出现，侧边栏的**未读小红点**帮你不漏消息。

- 💬 访问地址：[https://gfssm.butian.club/chat](https://gfssm.butian.club/chat)
- 左侧是频道列表，中间是消息，右侧（在宽屏上）是当前频道的成员。

## 频道体系

聊天分成几种「房间」，各有各的用途，登录后**该进的房间会自动帮你进好**，不用手动加入。

| 频道 | 谁在里面 | 用来干嘛 |
| --- | --- | --- |
| **公司总群** | 全体成员 | 全员级的公告、决策与跨部门协调——重要的事发这里 |
| **开放论坛** | 全体成员 | 自由讨论：想法、提问、分享，怎么聊都行，不怕打扰别人 |
| **部门频道** | 你所在的部门 | 本部门内部沟通，每个部门一个（结构建造 ST / 运营维护 OP / 人因 HF / 设施基建 IN / 进度成本 CD / 管理层）；频道 logo 显示部门两字母代码 |
| **私聊** | 你和某一个人 | 一对一的悄悄话 |

:::note[两个全员频道，分工不同]
**公司总群**是「正事」房间，**开放论坛**是「随便聊」房间。把半成品的想法丢进论坛，
不用纠结「这值不值得打扰所有人」；真正要全员知道的结论，再进总群。
:::

- **部门频道**随部门自动生成：新建一个部门，它的频道也会自动出现；部门成员自动加入。
- **管理员**会待在所有频道里（私聊除外）——因为他们需要看到全局。

### 开始一段私聊

想单独找某个人：在频道右侧的**成员列表**里点他的头像，或点某条消息作者旁的**私聊**按钮，
就会开出一个只有你俩的私聊房间。同一个人重复发起，进的是同一个房间，不会重复建。

## 置顶频道

聊天多了之后，重要的频道容易被刷下去。像微信一样，你可以**把频道置顶**：

- 在频道列表里把某个频道**置顶**，它就会固定排在最上面。
- 置顶是**你自己的**设置——每个人各自决定置顶什么，互不影响。

## 发消息、未读与静音

- 在底部输入框打字，回车发送；消息实时送达，别人无需刷新。
- 侧边导航的**聊天**入口带**未读计数**，有人找你一眼就能看到。
- 不想被某个频道频繁打扰？把它**静音**——消息还在，但不再计入未读提醒。

## @提及别人

在消息里用 `@` 点名，被点到的人会更容易注意到：

| 写法 | 效果 |
| --- | --- |
| `@某人` | 提醒某一位成员 |
| `@部门名` | 提醒某个部门的全体成员（如 `@结构建造部`） |
| `@所有人` | 提醒当前频道里的所有人（也可写 `@全体` / `@everyone`） |

输入 `@` 时会弹出候选列表，选一下即可，不用手打全名。

:::tip[@所有人 请省着用]
`@所有人` 会惊动整个频道，重要的事再用。日常找具体的人，`@他本人`或`@他的部门`就够了。
:::

## 投票、接龙与报名

聊天里可以直接发起轻量的小工具，收集意见或统计人数，不用另开表格：

| 工具 | 适合场景 | 怎么玩 |
| --- | --- | --- |
| **投票** | 多选一，比如定时间、选方案 | 发起后大家点选项，实时看票数 |
| **接龙** 🔗 | 每人接一行，比如报菜名、列清单 | 一个人可以接多行，内容自由填 |
| **报名** 📝 | 统计参加名单 | 每人一个名额，点一下加入 / 退出 |

发起入口在消息输入区，填好标题（接龙/报名再填第一条）即可。

## 把消息变成任务 / 工单 / 公告

聊天里聊出来的事，常常需要「落地」。把鼠标移到一条消息上，工具栏里可以**一键转化**：

| 按钮 | 效果 | 落到哪里 |
| --- | --- | --- |
| **转为任务** | 用这条消息开一个任务 | [任务](./tasks.md) 页面 |
| **转为工单** | 用这条消息开一个工单 | [工单](./tickets.md) 页面 |
| **转为公告** | 用这条消息发一条公告 | [公告](./announcements.md) / 首页 |

这样「谁说的、原话是什么」就不会在刷屏里丢掉，直接变成能跟进的条目。

:::note
消息里如果出现 `/tasks/...`、`/tickets/...`、`/docs/...` 这样的站内路径，会自动变成可点的链接，
方便顺手把人带到对应的任务、工单或文档。
:::

## 在频道里搜消息

「上周谁提过那个数」——不用一直往上翻：

1. 打开当前频道的**搜索**。
2. 输入关键词，在**本频道**范围内查找。
3. 还能进一步筛选：**按发言人**过滤，或只看**带附件**的消息。

## AI 助手（@ai）

平台内置了一个 AI 助手，它**只在聊天里出现**，不占用成员名单。用法很简单——在任意频道里
`@ai` 加上你的问题，它就会回复：

- 让它帮忙**梳理讨论、起草文字、解释概念、出主意**都可以。
- 它是当作一个普通成员来 `@` 的，所以在公司总群、部门频道、私聊里都能叫它。

:::tip
把 `@ai` 当成随叫随到的助手：`@ai 帮我把上面这段讨论总结成三条要点`。
:::

## 自动播报到公司总群

有些重要动态会**自动同步**到**公司总群**，让全员不用到处翻也能知道进展：

- **公告**发布后，会在公司总群里播报一条。
- **任务 / 工单的关键进展**（如新建、状态变化）也会汇报到公司总群。

这样总群就成了「全社动态的时间线」，重要的事都在一处过一遍。

## 常见问题

| 问题 | 说明 |
| --- | --- |
| 我要手动加入频道吗？ | 不用。该进的（总群、论坛、你的部门）都自动进好了 |
| 置顶会影响别人吗？ | 不会，置顶是你自己的视图设置 |
| `@所有人` 没生效？ | 确认写的是 `@所有人` / `@全体` / `@everyone`，并从候选里选中 |
| 找不到某人的私聊？ | 在成员列表点他头像发起，同一个人只会有一个私聊房间 |
| AI 怎么不在成员里？ | 它不是成员，只在聊天里用 `@ai` 呼叫 |
