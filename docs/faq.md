# 常见问题

这里用问答形式回答大家最常遇到的问题。找不到答案时，可以顺着每条里的链接深入，或直接联系社团管理员。

## 登录与账号

### Q：为什么我登不上？

A：常见有三种原因：

- **账号待审批（PENDING）**：注册后需要管理员审批，审批前登录会被引导到等待页。
- **账号被禁用（DISABLED）** 或被拒绝：无法登录，需要联系管理员。
- **密码错误**：核对用户名密码；仍不行请找管理员 **重置密码**。

详见 [注册与审批](./register.md) 与 [账号状态说明](./account-states.md)。

### Q：我注册了，为什么还进不去？

A：注册不等于开通。账号需要 **管理员审批** 通过后才能使用；如果一直没通过，可能是管理员还没处理，或平台当前 **关闭了注册**。请联系管理员确认，见 [注册与审批](./register.md)。

### Q：忘记密码怎么办？

A：平台 **没有自助找回密码** 功能。请直接联系社团管理员帮你重置，不要反复尝试登录。

### Q：能不能有多个账号？

A：可以。同一浏览器内 **最多保存 3 个账号**，通过顶栏的账号切换器快速切换，不用反复退出登录。详见 [访问与登录](./access.md)。

## 权限与后台

### Q：为什么我看不到管理后台？

A：管理后台只对 **管理员角色** 开放，普通成员看不到入口属于正常现象。而且后台现在**只保留系统级设置**（站点设置、倒计时、附件、操作日志）——任务、工单、文档、部门、成员、公告的管理都已**下放到各自的功能页面**，那些管理入口同样只有管理员可见。角色划分见 [角色与权限](./roles-permissions.md)。

### Q：我应该有某个功能，但打不开？

A：多数功能受权限码约束（例如成员目录的 `members:view`、工作日志的 `worklogs:view`）。缺权限时对应入口会不可见或打不开，可联系管理员核实，参考 [角色与权限](./roles-permissions.md)。

## 预览版与数据安全

### Q：登录页的「预览版」点了能干嘛？

A：那是一个 **免注册的演示沙盒**。不用账号就能进去随便点，体验平台的各个模块。它与正式环境 **完全隔离**，你在里面的任何操作都不影响真实数据。详见 [预览模式](./preview.md)。

### Q：预览会不会污染正式数据？我的数据安全吗？

A：不会。预览沙盒是 **完全隔离** 的独立环境，里面的增删改不会流入正式数据，正式数据也不会因为别人玩预览而改变。放心体验，见 [预览模式](./preview.md)。

## 使用与设置

### Q：手机能用吗？

A：可以。平台 **桌面端优先**，但同时兼容手机和平板，用手机浏览器即可打开 [https://gfssm.butian.club](https://gfssm.butian.club)。复杂操作（如编辑长文档）在电脑上体验更好。

### Q：怎么改主题和语言？

A：在 **外观设置** 里改。主题可选 浅色 / 深色 / 跟随系统，语言可选 中文 / 英文，还能自定义品牌主色。个人中心和登录页右上角都有外观控件，见 [个人中心与外观](./profile.md)。

### Q：为什么我没收到通知？

A：平台通知是 **纯站内** 的，不发邮件、不发短信。请回到平台看顶栏的铃铛图标是否有红点和未读计数；能收到哪些事件取决于你是否是相关人。详见 [通知中心](./notifications.md)。

### Q：在哪里记录我每天做了什么？

A：在个人中心（/profile）的 **工作日志** 里，用 Markdown 记录，可编辑可删除。见 [工作日志](./worklogs.md)。

### Q：怎么找到某位同伴、看他在不在线？

A：打开 [成员目录](./members.md)（/members），可以查看成员资料、部门、角色和在线状态。在线状态像 QQ 一样分为 **离线（灰，自动）/ 在线 / 空闲 / 工作中 / 忙碌**——想即时对接就找在线的人，看到忙碌就用 [聊天](./chat.md) 留言。

### Q：怎么和大家聊天？要手动加群吗？

A：打开侧边栏的 **聊天**（/chat）即可。该进的房间都**自动帮你进好**了：**公司总群**（全员正事）、**开放论坛**（全员闲聊）、你所在的**部门频道**；想单独找人就点头像开**私聊**。还能 @提及、发起投票 / 接龙 / 报名，把消息一键转成任务 / 工单 / 公告。详见 [聊天](./chat.md)。

### Q：平台的 AI 助手在哪？

A：AI 助手**只在聊天里**出现，不占成员名单。在任意频道里 `@ai` 加上你的问题，它就会回复——让它帮你总结讨论、起草文字、出主意都行。见 [聊天](./chat.md)。
