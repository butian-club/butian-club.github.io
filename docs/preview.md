# 预览模式（🧪 预览版）

如果你只是想快速体验平台，还没有邀请码、也不想注册，**预览模式** 就是为你准备的。它让你免注册进入一个 **隔离的演示沙盒**，在其中可以自由点击、任意试用，且 **绝不影响正式数据**。

## 一句话理解

在 [登录页](https://gfssm.butian.club/login) 点 **「🧪 预览版」**，系统就会为你临时开一个专属的演示环境，你以 **临时管理员** 身份进去，把后台的所有功能都体验一遍。

## 怎么进入

1. 打开登录页：[https://gfssm.butian.club/login](https://gfssm.butian.club/login)
2. 点击 **「预览版」**。
3. 系统自动为你创建一个专属沙盒，并把你带进去。
4. 开始体验。

无需用户名、密码，也无需注册或邀请码。

## 沙盒里有什么

进入预览时，系统会自动 **克隆一份 TEMPLATE 演示数据** 到你专属的沙盒，让平台看起来像已经在正常使用一样。克隆的演示内容包括：

| 演示数据 | 说明 |
| --- | --- |
| **用户** | 一批示例成员，方便体验成员目录、指派、在线状态等 |
| **部门** | 示例部门划分，可体验按部门指派与部门频道 |
| **聊天** | 公司总群 / 开放论坛 / 部门频道等，可体验发消息、@提及、投票、接龙 |
| **任务** | 示例任务，可查看、编辑、体验流转 |
| **工单** | 示例工单，可体验申请与处理 |
| **文档** | 示例文档，可体验阅读与编辑 |
| **公告** | 示例公告，可体验发布与置顶 |
| …… | 等等 |

你在预览里以 **临时管理员** 身份登录，因此可以体验 **全部管理功能**——包括平时只有管理员才能看到的管理入口（任务 / 工单 / 文档 / 部门 / 成员 / 公告的管理都**内联在各功能页面**上），以及后台保留的系统级设置（站点设置、倒计时、附件、操作日志）。想了解这些，可对照 [管理员指南](./overview.md) 边看边点。

## 完全隔离，放心操作

:::tip[怎么点都没关系]
预览沙盒是 **完全隔离** 的：你在里面的 **所有写入操作只影响你自己的沙盒**，绝不会影响平台的正式数据，也不会被别人看到。想删就删、想改就改，尽情尝试。
:::

## 心跳保活与自动销毁

预览沙盒是临时的，会在你离开后自动清理，机制如下：

- **心跳保活**：只要你的浏览器窗口开着，它 **每 25 秒** 会发一次心跳，告诉系统「我还在用」。
- **自动销毁**：当你 **关闭窗口** 后，心跳停止，大约 **60 秒** 后该沙盒会被自动销毁，里面的演示数据一并清除。
- **顶栏标记**：预览期间顶栏会显示 **「🧪 预览版」** 标记，提醒你当前身处沙盒。
- **主动退出**：点击顶栏的 **「🧪 预览版」** 标记，可以 **立即销毁沙盒并退出**。

:::warning[沙盒里的东西不会保留]
预览是一次性的体验环境。沙盒销毁后，你在里面创建或修改的一切都会消失，**无法找回**。不要把重要内容只放在预览沙盒里。
:::

## 限流规则

为了保证服务稳定、避免资源被占满，预览模式有两条限制：

| 限制类型 | 规则 |
| --- | --- |
| **单 IP 限流** | 每个 IP **每小时最多 5 次** 创建预览 |
| **全局并发** | 全平台 **同时最多 20 个** 预览沙盒 |

如果触及上限（例如全局已有 20 个沙盒在跑），系统会 **提示你稍后再试**。等一会儿，或稍后换个时间再进入即可。

## 预览模式适合谁

- 想先看看平台长什么样、再决定要不要注册的新同学。
- 想演示 / 介绍平台功能，又不想动用正式数据的人。
- 想安全地试验某个后台操作、确认效果的人。

如果你确定要长期使用平台，就应该正式 [注册与邀请码](./register.md)，用自己的账号登录，这样数据才会被真正保存。
