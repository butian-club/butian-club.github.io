# 首页与比赛倒计时

**首页**（`/`）是你登录后看到的第一个页面，也是整个协作平台的「入口大厅」。
它把最需要关注的信息——重要公告、比赛还剩多少时间、最近的任务和工单动态——
汇总在一屏里，让你不用逐个菜单点进去就能掌握当前进度。

- 🏠 访问地址：[https://gfssm.butian.club/](https://gfssm.butian.club/)
- 登录后会自动跳转到这里；点击顶部导航的 Logo 或「首页」也能随时回到这里。

:::tip[还没登录？]
首页需要登录后才能查看。如果你还没有账号或进不去，请先看
[访问平台](./access.md) 与 [注册与登录](./register.md)。
想先大致浏览，可以使用 [预览模式](./preview.md)。
:::

## 首页有哪些区块

| 区块 | 内容 | 数据来源 |
| --- | --- | --- |
| **欢迎语** | 可配置的问候语与站点名 | 管理员在站点设置里配置 |
| **置顶公告** | 被标记为置顶的公告 | [公告](./announcements.md)模块 |
| **比赛倒计时** | 距离目标日期还剩多少天 | 管理员在倒计时后台配置 |
| **任务概览动态** | 最近的任务进展 | [任务](./tasks.md)模块 |
| **工单概览动态** | 最近的工单动态 | [工单](./tickets.md)模块 |

:::note
欢迎语、站点名称等展示文案由管理员统一维护，你看到的内容取决于当前的站点配置。
配置方式见 [站点设置](./settings.md)。
:::

## 置顶公告

首页会优先展示被**置顶**的公告，方便所有成员第一时间看到社团的重要通知
（例如报名截止、集训安排、规则变更等）。

- 点击公告标题即可进入公告详情，阅读完整的 Markdown 正文。
- 只有管理员能发布公告和设置置顶；普通成员只读。**管理员就在首页直接发布 / 编辑 / 删除公告**，
  发布后还会自动播报到聊天的公司总群。详见 [公告](./announcements.md)。

## 比赛倒计时

比赛倒计时是首页最醒目的模块之一。它会根据管理员设定的**目标日期**自动倒数，
让全社对「还剩多少时间」有统一的紧迫感。

### 倒计时展示什么

- **比赛名称**：支持中文与英文两种名称。
- **目标日期**：倒计时的截止时间，首页据此自动计算剩余天数。
- **描述**：对这场比赛的补充说明。
- **开关**：管理员可以随时开启或关闭倒计时的展示。

### 谁来配置

比赛倒计时由管理员在后台（`/admin/countdown`）配置：填写比赛名称（中/英）、
目标日期、描述，并用开关控制是否在首页显示。配置完成后，首页会自动开始倒数，
无需成员做任何操作。

:::info
如果你是成员，发现倒计时信息不对（比如日期写错、还在显示已经结束的比赛），
请联系管理员到 `/admin/countdown` 更新，而不是自己改——成员没有配置权限。
:::

## 任务与工单动态

首页会汇总最近的**任务**与**工单**动态，帮你快速判断哪里需要跟进：

- **任务动态**：让你一眼看到最近有进展或临近截止的任务。点进去就是任务详情，
  可以查看描述、状态、指派与进展记录。完整用法见 [任务](./tasks.md)。
- **工单动态**：反映最近提交或处理中的工单。成员可以自己新建工单提资源申请
  或反馈问题，管理员负责处理。完整用法见 [工单](./tickets.md)。

:::tip[配合通知中心一起用]
首页给你的是「概览」，而 [通知中心](./notifications.md) 会在任务被指派给你、
工单有新评论等事件发生时主动提醒你，并带未读计数。两者搭配，既能纵览全局，
也不会漏掉和自己相关的动作。
:::

## 我能在首页做什么

首页主要用于**浏览和跳转**，本身不承载复杂操作。你可以：

1. 阅读置顶公告，掌握社团最新通知。
2. 查看比赛倒计时，了解距离截止还有多久。
3. 顺着任务 / 工单动态点进具体条目跟进。
4. 通过顶部导航进入任务、工单、文档、成员目录等各个模块。

你能看到多少内容、能做哪些操作，取决于你的角色。大多数成员是「查看 + 有限操作」，
创建、编辑、指派等管理动作通常需要管理员。角色与权限的完整说明见
[角色与权限](./roles-permissions.md)。

## 常见问题

- **首页一片空白 / 没有动态？** 可能是暂时还没有公告、任务或工单，属于正常现象；
  也可能是你的账号状态还未激活，参见 [账号状态](./account-states.md)。
- **倒计时不见了？** 管理员可能关闭了倒计时开关，或比赛尚未配置。
- **想改欢迎语 / 站点名？** 这些属于站点设置，见 [站点设置](./settings.md)。
